
Instalação de dependências
No diretório do projeto, instale as seguintes dependências:

npm init -y
npm install express mysql2 bcryptjs jsonwebtoken dotenv body-parser

se estiver usando a versão atual para importações tire o const nomeda_biblioteca = require ('nomeda_biblioteca'); e mude para import nome da biblioteca from 'nome da biblioteca';
